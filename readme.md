# GTI — Proyecto Web

Este es un proyecto semestral que estoy desarrollando para la materia de Proyecto Web en el Grado en Tecnologías Interactivas (GTI) en el Campus de Gandia de la UPV. Es un entorno de desarrollo propio para aplicar lo aprendido en programación web y bases de datos.

</div>

---

## El Proyecto

El sistema se divide en dos partes: la empresa que vende el software (**GTI**) y el producto educativo (**DOA**).

* **Landing Page (GTI):** Es la página de la empresa donde se muestran los productos que ofrecemos. Para poder comprar o probar la beta de nuestro producto principal (DOA), el cliente tiene que registrarse e iniciar sesión en esta plataforma.
* **Plataforma Educativa (DOA):** Al acceder a la beta, te lleva al inicio de sesión de DOA, el cual cuenta con tres dashboards independientes según el rol:
  * **Administrador:** Crea grados, cursos y asignaturas. Asigna profesores a sus materias y alumnos a sus grados/asignaturas de forma flexible (sin importar el curso). También gestiona anuncios globales.
  * **Profesor:** Panel para ver a sus alumnos asignados. Puede subir, editar o borrar tareas, exámenes, recursos y anuncios específicos de las asignaturas que imparte.
  * **Alumno:** Vista de sus asignaturas con sus respectivos exámenes, tareas, recursos, calendario, anuncios y un módulo unificado para ver sus notas.
